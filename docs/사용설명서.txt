======================================
  은지의 블로그 자동 포스트 도구
  사용설명서
======================================


【 파일 구성 】
─────────────────────────────────────
  은지의블로그.exe   ← 실행 파일 (더블클릭)
  data\              ← 글쓰기 스타일 데이터 (건드리지 마세요)
  .env               ← 계정 정보 (최초 1회 설정)
  사용설명서.txt     ← 이 파일


════════════════════════════════════
  1단계. 최초 설정 (처음 한 번만)
════════════════════════════════════

1. 은지의블로그.exe 를 더블클릭해서 실행합니다.
   (처음 실행 시 Chromium 브라우저 자동 설치 - 잠깐 기다리세요)

2. 상단 탭에서 [설정] 탭을 클릭합니다.

3. 아래 정보를 입력합니다:
   - 네이버 ID  : 네이버 아이디 (예: eunjikim)
   - 네이버 PW  : 네이버 비밀번호
   - 블로그 ID  : 블로그 주소의 ID (예: punx2-)
   - 카테고리   : 맛집/카페 (기본값 유지)

4. [저장] 버튼을 클릭합니다.

※ Gemini API 키는 .env 파일에 이미 설정되어 있습니다.
   GEMINI_API_KEY= 뒤에 우찬이가 준 키를 붙여넣으세요.


════════════════════════════════════
  2단계. 사진 폴더 준비하기
════════════════════════════════════

1. 아무 폴더나 만들어서 가게/장소 이름으로 이름 붙이기
   예) C:\사진\2026-05-20 홍대 스타벅스

2. 그 폴더 안에 찍은 사진들을 넣기
   - 카카오톡에서 받은 jpg 파일 그대로 넣으면 됩니다
   - HEIC 파일은 jpg로 변환 후 넣어주세요


════════════════════════════════════
  3단계. 블로그 글 자동 생성하기
════════════════════════════════════

1. [글 작성] 탭에서 [찾아보기] 버튼 클릭
2. 아까 만든 사진 폴더를 선택합니다
3. 카테고리를 선택합니다 (맛집/카페 / 패션 / 뷰티)
4. 정보를 입력합니다:
   - 카페/음식점 이름 : 가게 이름 (예: 스타벅스 홍대점)
   - 위치             : 동네 이름 (예: 홍대)
   - 방문일           : 방문한 날짜 (예: 2026-05-20)
   - 메뉴             : 시킨 메뉴와 가격 (예: 아메리카노 5000원)
   - 한줄평           : 솔직한 감상 (예: 분위기 좋고 조용함)
5. [내돈내산 / 협찬] 선택
6. [✨ 글 생성하기] 버튼 클릭

→ Gemini AI가 사진을 보고 블로그 글을 써줍니다 (30초~1분)
→ 완료되면 로그창에 제목과 태그가 표시됩니다


════════════════════════════════════
  4단계. 네이버 블로그에 자동 포스팅
════════════════════════════════════

1. 글 생성이 완료되면 [🚀 네이버에 올리기] 버튼이 활성화됩니다
2. 버튼을 클릭하면 크롬 브라우저가 자동으로 열립니다

   ┌──────────────────────────────────────────┐
   │ 처음 사용 시: 네이버 로그인 화면이 뜹니다  │
   │ 직접 로그인한 후, 프로그램 창에서         │
   │ [확인] 버튼을 누르세요.                   │
   │ (두 번째부터는 자동 로그인됩니다)          │
   └──────────────────────────────────────────┘

3. 자동으로 글 제목, 본문, 사진, 태그, 장소가 입력됩니다
4. 브라우저에서 글 내용을 확인하고 [발행] 버튼을 직접 눌러주세요
5. 확인이 끝나면 브라우저 창을 닫으면 됩니다


════════════════════════════════════
  자주 묻는 질문
════════════════════════════════════

Q. 프로그램이 안 열려요
A. 바이러스 백신이 차단하는 경우가 있습니다.
   "위협 허용" 또는 "예외 추가" 하고 다시 실행해보세요.

Q. 로그인이 안 돼요
A. data 폴더 안의 auth_state.json 파일을 삭제하고 다시 시도하세요.

Q. 사진이 안 올라가요
A. 사진 파일명에 한글이 없는지 확인하세요.
   카카오톡에서 보낸 그대로의 파일명이 가장 안전합니다.

Q. 글 제목에 주소가 들어가요
A. 자동으로 제거되도록 되어 있습니다.
   혹시 남아있으면 브라우저에서 직접 수정해주세요.

Q. Chromium 설치가 너무 오래 걸려요
A. 최초 1회만 설치합니다. Wi-Fi 연결 상태를 확인하세요.
